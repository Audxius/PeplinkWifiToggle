# PeplinkWifiToggle
Wifi toggle based on speed for PEPLINK routers (docker)

------------------------------------------------------------------------

build with:
docker buildx build   --platform linux/amd64,linux/arm64,linux/arm/v7   -t audriuspeplink/peplinkgpswifishutdown:stable   --push .

------------------------------------------------------------------------
